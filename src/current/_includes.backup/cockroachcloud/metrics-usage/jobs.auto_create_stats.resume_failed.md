This metric is a high-level indicator that automatically generated [table statistics]({% link {{ site.current_cloud_version }}/cost-based-optimizer.md %}#table-statistics) is failing. Failed statistic creation can lead to the query optimizer running with stale statistics. Stale statistics can cause suboptimal query plans to be selected leading to poor [query performance]({% link {{ site.current_cloud_version }}/performance-best-practices-overview.md %}).
