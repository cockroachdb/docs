This page explains the prerequisites for using [MOLT Fetch]({% link molt/molt-fetch.md %}) and then describes how to install it.

## Supported databases

The following source databases are supported:

- PostgreSQL 11-16
- MySQL 5.7-8.4
- Oracle Database 19c (Enterprise Edition) and 21c (Express Edition)
