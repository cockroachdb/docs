## July 24, 2023

<h3> `ccloud` changes </h3>

- `ccloud` [v0.5.11](.../cockroachcloud/ccloud-get-started.html#install-ccloud) is now available. This update includes a new [`--skip-ip-check` flag](../cockroachcloud/ccloud-get-started.html#skip-the-ip-allowlist-check-when-connecting-to-your-cluster) that allows users to skip the client-side IP allowlist check when connecting to a cluster using the `ccloud cluster sql` command.
