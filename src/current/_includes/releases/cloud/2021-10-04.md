## October 4, 2021

<h3>General changes</h3>

- New CockroachCloud clusters will now run CockroachDB [v21.1.9]({% link releases/v21.1.md %}#v21-1-9).

<h3>Bug fixes</h3>

- Fixed an error in the connection string for Windows users [connecting to CockroachCloud Free (beta)](https://www.cockroachlabs.com/docs/cockroachcloud/connect-to-a-serverless-cluster) clusters.

<h3>Miscellaneous changes</h3>

- Cluster names are now included in cluster creation email alerts.
