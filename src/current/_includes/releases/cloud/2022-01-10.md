## January 10, 2022

<h3>General changes</h3>

- New {{ site.data.products.dedicated }} clusters will now run [v21.2.3]({% link releases/v21.2.md %}#v21-2-3).
- {{ site.data.products.serverless }} clusters will now run CockroachDB [v21.2.0-beta.4]({% link releases/v21.2.md %}#v21-2-0-beta-4).
- The CockroachDB documentation navigation is now organized by user task instead of by product for {{ site.data.products.serverless }}, {{ site.data.products.dedicated }}, and {{ site.data.products.core }} v21.2. Topics specific to Serverless and Dedicated clusters are within the new top-level user task categories. {{ site.data.products.db }} release notes are under Reference.

<h3>Console changes</h3>

- The [**Billing**](https://www.cockroachlabs.com/docs/cockroachcloud/billing-management) page is now separated into two tabs, **Overview** and **Payment Details**.
