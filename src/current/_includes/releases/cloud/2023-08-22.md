## August 22, 2023

<h3 id="2023-08-16-general-changes"> General changes </h3>

- In the {{ site.data.products.cloud }} Console, you can now [add regions and change the primary region]({% link cockroachcloud/folders.md %}) for multi-region  CockroachDB {{ site.data.products.basic }} (named Serverless at the time of this release) clusters. You cannot currently edit the region configuration for a single-region cluster once it has been created, and you cannot remove a region once it has been added. For details, refer to [Planning a CockroachDB {{ site.data.products.basic }} Cluster]({% link cockroachcloud/plan-your-cluster-basic.md %}#multi-region-clusters).
