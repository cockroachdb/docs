<h3 id="v24-3-28-bug-fixes">Bug fixes</h3>

- Fixed a bug where context chain growth was directly proportional to the `cockroach` process uptime. In rare cases, where the process had been running for several months, a race condition could arise where the runtime copying a goroutines stack due to the long context chain could intersect with a CPU profiling request, resulting in what appeared to be a stalled process. [#164778][#164778]
