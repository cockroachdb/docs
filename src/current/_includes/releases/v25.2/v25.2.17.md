## v25.2.17

Release Date: April 20, 2026

{% include releases/new-release-downloads-docker-image.md release=include.release %}

<h3 id="v25-2-17-bug-fixes">Bug fixes</h3>

- Fixed a bug where concurrent updates to a table using multiple column families during a partial index creation could result in data loss, incorrect `NULL` values, or validation failures in the resulting index. [#168473][#168473]
- Fixed a bug that caused Logical Data Replication (LDR) job creation to fail during rolling upgrades from v24.3 directly to v25.2. [#167433][#167433]
