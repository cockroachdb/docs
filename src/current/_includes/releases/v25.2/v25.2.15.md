<h3 id="v25-2-15-bug-fixes">Bug fixes</h3>

- Fixes a bug in kafka v1 where a changefeed
  could potentially hang if the changefeed was shutting down. [#162118][#162118]
- Fixes a bug where generating a debug zip causes
  a node to OOM. This OOM happens when a log file contains a malformed
  log and is using json (or json-compact) formatting. [#163357][#163357]
