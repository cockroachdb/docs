{{site.data.alerts.callout_danger}}On Microsoft Azure, A-series VMs can come back from VM migrations with bad filesystems, which can in turn cause data corruption and performance degradation. Before using any other Azure VM series in production, you should test them extensively.{{site.data.alerts.end}}
