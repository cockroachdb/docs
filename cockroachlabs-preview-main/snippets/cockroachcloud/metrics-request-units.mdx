<Note>
These graphs are available for CockroachDB Basic deployments. For graphs available to CockroachDB Standard or Advanced deployments, refer to the <a href="/docs/VERSION_PLACEHOLDER/cockroachcloud/metrics#cockroachdb-cloud-console-metrics-page" data-versioned="true">CockroachDB Cloud Console Metrics page</a>.
</Note>

The Request Unit metrics let you monitor <a href="/docs/VERSION_PLACEHOLDER/cockroachcloud/resource-usage-basic" data-versioned="true">resource consumption</a>. All cluster activity, including <a href="/docs/VERSION_PLACEHOLDER/selection-queries" data-versioned="true">SQL queries</a>, bulk operations, and <a href="/docs/VERSION_PLACEHOLDER/show-jobs" data-versioned="true">background jobs</a>, is measured in Request Units, or RUs. An RU is an abstracted metric that represents the compute and I/O resources used by a database operation. In addition to queries that you run, background activity, such as <a href="/docs/VERSION_PLACEHOLDER/cost-based-optimizer#table-statistics" data-versioned="true">automatic statistics</a> to optimize your queries or connecting a <a href="/docs/VERSION_PLACEHOLDER/change-data-capture-overview" data-versioned="true">changefeed</a> to an external sink, also consumes RUs.

To view these graphs, select a cluster from the <a href="/docs/VERSION_PLACEHOLDER/cockroachcloud/cluster-management#view-clusters-page" data-versioned="true">**Clusters** page</a>, and click **Metrics** in the **Monitoring** section of the left side navigation. On the **Metrics** page, click the **Request Units** tab.

## Time interval selection

The time interval selector at the top of each tab allows you to filter the view for a predefined or custom time interval. Use the navigation buttons to move to the previous, next, or current time interval. When you select a time interval, the same interval is selected for all charts on the **Metrics** page.
