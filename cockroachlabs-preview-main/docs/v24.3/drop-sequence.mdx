The `DROP SEQUENCE` [statement](/docs/v24.3/sql-statements) removes a sequence from a database.

<Note>
The `DROP SEQUENCE` statement performs a schema change. For more information about how online schema changes work in CockroachDB, see [Online Schema Changes](/docs/v24.3/online-schema-changes).
</Note>

## Required privileges

The user must have the `DROP` [privilege](/docs/v24.3/security-reference/authorization#managing-privileges) on the specified sequence(s).

## Examples

### Remove a sequence (no dependencies)

In this example, other objects do not depend on the sequence being dropped.

```sql
> CREATE SEQUENCE even_numbers INCREMENT 2 START 2;
```

```sql
> SHOW SEQUENCES;
```

```text
  sequence_schema | sequence_name
------------------+----------------
  public          | even_numbers
(1 row)
```

```sql
> DROP SEQUENCE even_numbers;
```

```sql
> SHOW SEQUENCES;
```

```text
  sequence_schema | sequence_name
------------------+----------------
(0 rows)
```
