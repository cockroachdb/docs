title: "Staged Release Process"
old_description: "Learn about Cockroach Labs' staged release process for CockroachDB Cloud and Self-Hosted releases."
---

As of 2024, CockroachDB uses a staged release process. New versions roll out to select CockroachDB Cloud organizations first. CockroachDB self-hosted binaries are published later, with timing based on release type:

- Major releases (x.y.0 GA): Approximately 2 weeks after Cloud GA.
- Patch releases (x.y.1+): Approximately 1 week after Cloud availability.

Exact timing may vary based on rollout safety and scheduling.
