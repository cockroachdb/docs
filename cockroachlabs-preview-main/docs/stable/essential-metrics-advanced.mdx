These essential CockroachDB metrics let you monitor your CockroachDB Advanced cluster. Use them to build custom dashboards with the following tools:

- [Datadog integration](/docs/stable/cockroachcloud/tools-page#monitor-cockroachdb-cloud-with-datadog) - The [**Datadog Integration Metric Name**](https://docs.datadoghq.com/integrations/cockroach-cloud#metrics) column lists the corresponding Datadog metric which requires the `crdb_dedicated` prefix.
- [Metrics export](/docs/stable/cockroachcloud/export-metrics-advanced)

The **Usage** column explains why each metric is important to visualize and how to make both practical and actionable use of the metric in a production deployment.
