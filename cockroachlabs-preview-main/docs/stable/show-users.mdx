The `SHOW USERS` [statement](/docs/v26.1/sql-statements) lists the users for all databases.

<Note>
Since the keywords `ROLES` and `USERS` can now be used interchangeably in SQL statements for enhanced PostgreSQL compatibility, `SHOW USERS` is now an alias for [`SHOW ROLES`](/docs/v26.1/show-roles).
</Note>

## Required privileges

The user must have the [`SELECT`](/docs/v26.1/select-clause) [privilege](/docs/v26.1/security-reference/authorization#managing-privileges) on the `system.users` and `system.role_members` tables.

## Example

```sql
> SHOW USERS;
```
