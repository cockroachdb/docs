The `DROP PROCEDURE` [statement](/docs/v26.1/sql-statements) drops a [stored procedure](/docs/v26.1/stored-procedures).

## Required privileges

To drop a procedure, a user must have the `DROP` [privilege](/docs/v26.1/security-reference/authorization#managing-privileges) on the procedure.

## Examples

### Drop a stored procedure

The following statement drops the [`delete_earliest_histories` example procedure](/docs/v26.1/stored-procedures#create-a-stored-procedure-using-pl-pgsql):

```sql
DROP PROCEDURE delete_earliest_histories;
```
