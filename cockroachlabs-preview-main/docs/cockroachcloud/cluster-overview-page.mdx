The **Cluster Overview** page displays key metrics about your CockroachDB Cloud cluster. To view this page, click on a cluster from the [**Clusters** page](/docs/cockroachcloud/cluster-management#view-clusters-page).

<Tabs>
<Tab title="CockroachDB Standard / Basic">

## Cluster settings

The **Cluster settings** panel shows the settings selected during [cluster creation](/docs/cockroachcloud/create-a-basic-cluster), as well as any [labels](/docs/cockroachcloud/labels) added to the cluster after its creation.

<table><thead><tr><th>Field</th><th>Description</th></tr></thead><tbody><tr><td>Cloud</td><td>The cluster's <a href="/docs/cockroachcloud/create-a-basic-cluster#step-2-select-the-cloud-provider">cloud provider</a>.</td></tr><tr><td>Plan type</td><td>The plan type used to create the cluster.</td></tr><tr><td>Regions</td><td>The cluster's <a href="/docs/cockroachcloud/create-a-basic-cluster#step-3-select-the-regions">regions</a>.</td></tr><tr><td>Labels</td><td>The <a href="/docs/cockroachcloud/labels">labels</a> added to the cluster.</td></tr></tbody></table>

## Capacity used this month

The **Capacity used this month** panel displays your cluster usage statistics for the current month. You can click [**Estimate usage cost**](/docs/cockroachcloud/basic-cluster-management#estimate-usage-cost) to open a tool that estimates your monthly costs based on your workload during a selected time frame.

<table><thead><tr><th>Field</th><th>Description</th></tr></thead><tbody><tr><td>Capacity used</td><td>The CockroachDB Standard or Basic cluster's <a href="/docs/cockroachcloud/create-a-basic-cluster#step-4-configure-cluster-capacity">configured capacity</a>. Click the <strong>edit</strong> icon to change the configured capacity.</td></tr><tr><td><a href="/docs/cockroachcloud/plan-your-cluster-basic#request-units">Request Units</a></td><td>The CPU and I/O resources being used by queries on the cluster, and whether throttling is in effect. The total available RUs are determined by your configured capacity.</td></tr><tr><td>Storage</td><td>The amount of data currently stored in the cluster. This value does not account for compression or replication. The total available storage is determined by your configured capacity.</td></tr></tbody></table>

## Current activity panel

This panel displays operational statistics for your cluster.

<table><thead><tr><th>Field</th><th>Description</th></tr></thead><tbody><tr><td>Throughput</td><td>The cluster throughput in queries per second (QPS).</td></tr><tr><td>P99 latency</td><td>The 99th percentile of service latency. This value indicates the time within which the cluster executed 99 percent of queries over the last minute.</td></tr><tr><td>Open sessions</td><td>The number of open sessions in the cluster.</td></tr></tbody></table>

## Cluster Overview metrics

These time-series graphs display important cluster metrics over time.

You can use the timeframe selector above the graphs to view data for a specific time period.

### Request Units

The **Request Units** graph displays the CPU and I/O resources being used by queries on the cluster. This is measured in Request Units per second at each timestamp. Simple queries consume few RUs, while complicated queries with many reads and writes consume more RUs.

### Storage Used

The **Storage Used** graph displays the amount of data being stored in the cluster over time. This is the logical number of live bytes and does not account for compression or replication.

### SQL Statements

The **SQL Statements** graph displays an average of the number of [`SELECT`](/docs/v25.4/select-clause), [`INSERT`](/docs/v25.4/insert), [`UPDATE`](/docs/v25.4/update), and [`DELETE`](/docs/v25.4/delete) statements successfully executed per second.

## PCI ready (with Security add-on)

CockroachDB Advanced clusters with Security add-on have a **PCI ready** panel to monitor the status of security features required for [PCI readiness](/docs/cockroachcloud/pci-dss). Feature statuses will update from **INACTIVE** to **ACTIVE** once you configure them. Learn more about configuring these features:

- [CockroachDB Cloud Organization Audit logs](/docs/cockroachcloud/cloud-org-audit-logs)
- [Cluster audit log export](/docs/cockroachcloud/export-logs)
- [Customer-Managed Encryption Keys (CMEK)](/docs/cockroachcloud/managing-cmek)
- [Egress Perimeter Controls](/docs/cockroachcloud/egress-perimeter-controls)
- Single Sign-On (SSO) for your [CockroachDB Cloud organization](/docs/cockroachcloud/configure-cloud-org-sso) and your [clusters](/docs/cockroachcloud/cloud-sso-sql)
- [Network security](/docs/cockroachcloud/network-authorization)
