The `DROP FUNCTION` [statement](/docs/v23.2/sql-statements) removes one or more [user-defined functions](/docs/v23.2/user-defined-functions) from a database.

<Note>
The `DROP FUNCTION` statement performs a schema change. For more information about how online schema changes work in CockroachDB, see [Online Schema Changes](/docs/v23.2/online-schema-changes).
</Note>

## Required privileges

To drop a function, a user must have the `DROP` [privilege](/docs/v23.2/security-reference/authorization#managing-privileges) on the function.
