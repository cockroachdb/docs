The statements on this page are **deprecated** as of v23.1 and will be removed in a future release. To move data into CockroachDB, use [`IMPORT INTO`](/docs/v23.2/import-into) or [`COPY FROM`](/docs/v23.2/copy).
</Warning>

The `cockroach import` [command](/docs/v23.2/cockroach-commands) imports a database or table from a local dump file into a running cluster. This command [uploads a userfile](/docs/v23.2/cockroach-userfile-upload), imports its data, then [deletes the userfile](/docs/v23.2/cockroach-userfile-delete). `PGDUMP` and `MYSQLDUMP` file formats are currently supported.

## Required privileges

The user must have `CREATE` [privileges](/docs/v23.2/security-reference/authorization#managing-privileges) on `defaultdb`.

## Synopsis

Import a database:

```shell
$ cockroach import db <format> <location/of/file> <flags>
```

Import a table:

```shell
$ cockroach import table <table_name> <format> <location/of/file> <flags>
```

View help:

```shell
$ cockroach import --help
```

## Supported Formats

- [`pgdump`](/docs/molt/migrate-to-cockroachdb)
- [`mysqldump`](/docs/molt/migrate-to-cockroachdb?filters=mysql)

## Examples

### Import a database

To import a database from a local file:

```shell
$ cockroach import db mysqldump /Users/maxroach/Desktop/test-db.sql --certs-dir=certs
```

```text
successfully imported mysqldump file /Users/maxroach/Desktop/test-db.sql
```

### Import a table

To import a table from a local file:

```shell
$ cockroach import table test_table pgdump /Users/maxroach/Desktop/test-db.sql --certs-dir=certs
```

```text
successfully imported table test_table from pgdump file /Users/maxroach/Desktop/test-db.sql
```

### Import a database with unsupported SQL syntax and log all unsupported statements

To import a database from a `PGDUMP` file that contains unsupported SQL syntax and log the ignored statements to a [userfile](/docs/v23.2/use-userfile-storage):

```shell
$ cockroach import db pgdump /Users/maxroach/Desktop/test-db.sql --certs-dir=certs --ignore-unsupported-statements=true --log-ignored-statements='userfile://defaultdb.public.userfiles_root/unsupported-statements.log'
```

```text
successfully imported table test_table from pgdump file /Users/maxroach/Desktop/test-db.sql
```

### Import a limited number of rows from a dump file

To limit the number of rows imported from a dump file:

```shell
$ cockroach import table test_table pgdump /Users/maxroach/Desktop/test-db.sql --certs-dir=certs --row-limit='50'
```

```text
successfully imported table test_table from pgdump file /Users/maxroach/Desktop/test-db.sql
```
