This page has instructions for making SQL [selection queries](/docs/v23.2/selection-queries) against CockroachDB from various programming languages.

## Before you begin

Before reading this page, do the following:

- [Create a CockroachDB Standard cluster](/docs/cockroachcloud/quickstart) or [start a local cluster](/docs/cockroachcloud/quickstart?filters=local).
- [Install a Driver or ORM Framework](/docs/v23.2/install-client-drivers).
- [Connect to the database](/docs/v23.2/connect-to-the-database).
- [Insert data](/docs/v23.2/insert-data) that you now want to run queries against.

<Note>
Your application should [use a retry loop to handle transaction errors](/docs/v23.2/query-behavior-troubleshooting#transaction-retry-errors) that can occur under [contention](/docs/v23.2/performance-best-practices-overview#understanding-and-avoiding-transaction-contention).
</Note>

## Simple selects

<Tabs>
<Tab title="SQL">

```sql
SELECT id, balance from accounts;
```

For more information about how to use the built-in SQL client, see the [`cockroach sql`](/docs/v23.2/cockroach-sql) reference docs.

</Tab>
<Tab title="Go">

```go
// 'db' is an open database connection

rows, err := db.Query("SELECT id, balance FROM accounts")
if err != nil {
    log.Fatal(err)
}
defer rows.Close()
fmt.Println("Initial balances:")
for rows.Next() {
    var id, balance int
    if err := rows.Scan(&id, &balance); err != nil {
        log.Fatal(err)
    }
    fmt.Printf("%d %d\n", id, balance)
}
```

For complete examples, see:

- [Build a Go App with CockroachDB](/docs/v23.2/build-a-go-app-with-cockroachdb) (pgx)
- [Build a Go App with CockroachDB and GORM](/docs/v23.2/build-a-go-app-with-cockroachdb)

</Tab>
<Tab title="Java">

```java
// ds is an org.postgresql.ds.PGSimpleDataSource

try (Connection connection = ds.getConnection()) {
    Statement stmt = connection.createStatement();
    ResultSet rs = stmt.executeQuery("SELECT id, balance FROM accounts");

    while (rs.next()) {
        int id = rs.getInt(1);
        int bal = rs.getInt(2);
        System.out.printf("ID: %10s\nBalance: %5s\n", id, bal);
    }
    rs.close();

} catch (SQLException e) {
    System.out.printf("sql state = [%s]\ncause = [%s]\nmessage = [%s]\n",
                      e.getSQLState(), e.getCause(), e.getMessage());
}
```

For complete examples, see:

- [Build a Java App with CockroachDB](/docs/v23.2/build-a-java-app-with-cockroachdb) (JDBC)
- [Build a Java App with CockroachDB and Hibernate](/docs/v23.2/build-a-java-app-with-cockroachdb-hibernate)

</Tab>
<Tab title="Python">

```python
# conn is a psycopg2 connection

with conn.cursor() as cur:
    cur.execute("SELECT id, balance FROM accounts")
    rows = cur.fetchall()
    for row in rows:
        print([str(cell) for cell in row])
```

For complete examples, see:

## Joins

The syntax for a [selection query](/docs/v23.2/selection-queries) with a two-way [join](/docs/v23.2/joins) is shown below.

```sql
SELECT
    a.col1, b.col1
FROM
    some_table AS a
    JOIN
    some_other_table AS b
    ON
    a.id = b.id
WHERE
    a.col2 > 100 AND a.col3 > now()
ORDER BY
    a.col2 DESC
LIMIT
    25;
```

Join performance can be a big factor in your application's performance. For more information about how to make sure your SQL performs well, see [Optimize Statement Performance](/docs/v23.2/make-queries-fast).
