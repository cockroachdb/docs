The following parameters are included for PostgreSQL compatibility and do not affect how CockroachDB runs:

- `autovacuum_enabled`
- `fillfactor`

For the list of storage parameters that affect how [Row-Level TTL](/docs/v23.2/row-level-ttl) works, see the list of [TTL storage parameters](/docs/v23.2/row-level-ttl#ttl-storage-parameters).

## Required privileges

The user must be a member of the [`admin`](/docs/v23.2/security-reference/authorization#roles) or [owner](/docs/v23.2/security-reference/authorization#object-ownership) roles, or have the [`CREATE` privilege](/docs/v23.2/security-reference/authorization#supported-privileges) on the table.

## Examples

### Create a table with row-level TTL enabled

```sql
CREATE TABLE ttl_test (
  id UUID PRIMARY KEY default gen_random_uuid(),
  description TEXT,
  inserted_at TIMESTAMP default current_timestamp()
) WITH (ttl_expire_after = '3 months');
```

```sql
SHOW CREATE TABLE ttl_test;
```

```text
  table_name |                                                                                         create_statement
-------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  ttl_test   | CREATE TABLE public.ttl_test (
             |     id UUID NOT NULL DEFAULT gen_random_uuid(),
             |     description STRING NULL,
             |     inserted_at TIMESTAMP NULL DEFAULT current_timestamp():::TIMESTAMP,
             |     crdb_internal_expiration TIMESTAMPTZ NOT VISIBLE NOT NULL DEFAULT current_timestamp():::TIMESTAMPTZ + '3 mons':::INTERVAL ON UPDATE current_timestamp():::TIMESTAMPTZ + '3 mons':::INTERVAL,
             |     CONSTRAINT ttl_test_pkey PRIMARY KEY (id ASC)
             | ) WITH (ttl = 'on', ttl_expire_after = '3 mons':::INTERVAL, ttl_job_cron = '@hourly')
(1 row)
```

In this case, CockroachDB implicitly added the `ttl` and `ttl_job_cron` [TTL storage parameters](/docs/v23.2/row-level-ttl#ttl-storage-parameters).
