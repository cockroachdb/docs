<InlineImage alt="Megaphone" src="/images/common/icon-megaphone.png" /> New in v23.2: The `DROP PROCEDURE` [statement](/docs/v23.2/sql-statements) drops a [stored procedure](/docs/v23.2/stored-procedures).

## Required privileges

To drop a procedure, a user must have the `DROP` [privilege](/docs/v23.2/security-reference/authorization#managing-privileges) on the procedure.

## Examples

### Drop a stored procedure

The following statement drops the [`delete_earliest_histories` example procedure](/docs/v23.2/stored-procedures#create-a-stored-procedure-using-pl-pgsql):

```sql
DROP PROCEDURE delete_earliest_histories;
```
