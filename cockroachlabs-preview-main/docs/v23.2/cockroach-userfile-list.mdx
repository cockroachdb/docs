The `cockroach userfile list` [command](/docs/v23.2/cockroach-commands) lists the files stored in the [user-scoped file storage](/docs/v23.2/use-userfile-storage) which match the [provided pattern](/docs/v23.2/cockroach-userfile-upload#file-destination), using a SQL connection. If no pattern is provided, all files in the specified (or default, if unspecified) user scoped file storage will be listed.

## Required privileges

The user must have the `CREATE` [privilege](/docs/v23.2/security-reference/authorization#managing-privileges) on the target database. CockroachDB will proactively grant the user `GRANT`, `SELECT`, `INSERT`, `DROP`, `DELETE` on the metadata and file tables.

A user can only view files in their own user-scoped storage, which is accessed through the [userfile URI](/docs/v23.2/cockroach-userfile-upload#file-destination) used during the upload. CockroachDB will revoke all access from every other user in the cluster except users in the `admin` role.

## Synopsis

View files:

```shell
$ cockroach userfile list <file | dir> [flags]
```

View help:

```shell
$ cockroach userfile list --help
```

## Examples

### List all files in the default storage

If the file or directory is not specified, all files in the default user-scoped storage (`userfile://defaultdb.public.userfiles_$user/`) will be listed:

```shell
$ cockroach userfile list --certs-dir=certs
```

```text
userfile://defaultdb.public.userfiles_root/test-data-2.csv
userfile://defaultdb.public.userfiles_root/test-data.csv
userfile://defaultdb.public.userfiles_root/test-upload/test-data.csv
```

### List a specific file

To list all files in a specified directory:

```shell
$ cockroach userfile list test-data.csv --certs-dir=certs
```

```text
userfile://defaultdb.public.userfiles_root/test-data.csv
```

### List files that match the provided pattern

To list all files that match a pattern, use `*`:

```shell
$ cockroach userfile list '*.csv' --certs-dir=certs
```

```text
userfile://defaultdb.public.userfiles_root/test-data-2.csv
userfile://defaultdb.public.userfiles_root/test-data.csv
```

### List files from a non-default userfile URI

If you [uploaded a file to a non-default userfile URI](/docs/v23.2/cockroach-userfile-upload#upload-a-file-to-a-non-default-userfile-uri) (e.g., `userfile://testdb.public.uploads`):

```shell
cockroach userfile upload /Users/maxroach/Desktop/test-data.csv userfile://testdb.public.uploads/test-data.csv
```

Use the same URI to view it:

```shell
cockroach userfile list userfile://testdb.public.uploads
```
