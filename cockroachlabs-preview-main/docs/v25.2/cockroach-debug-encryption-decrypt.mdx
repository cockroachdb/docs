<Warning>
We strongly recommend only using `cockroach debug encryption-decrypt` when working directly with the [Cockroach Labs support team](/docs/v25.2/support-resources).
</Warning>

The [`cockroach debug encryption-decrypt`](/docs/v25.2/cockroach-commands) command decrypts store files on a node that are [encrypted at rest](/docs/v25.2/encryption) so that they can be examined to add more context to logs gathered from the [`cockroach debug zip`](/docs/v25.2/cockroach-debug-zip) command. Unless they are decrypted, Cockroach Labs cannot examine store files that are encrypted at rest.

## Synopsis

```shell
cockroach debug encryption-decrypt {store_directory} {input_file} {output_file} --enterprise-encryption={encryption_spec} {flags}
```

Replace:

- `{encryption_spec}`: The cluster's [encryption details](/docs/v25.2/encryption#starting-a-node-with-encryption).
- `{store_directory}`: The directory where the node [stores data files](/docs/v25.2/cockroach-start#store).
- `{input_file}`: The name of an encrypted store file.
- `{output_file}`: A name for the decrypted store file. If not specified, the command will output the decrypted contents to `stdout`.

Run the command locally on a node. It does not support the `--url` flag.

After running the command, you can send the requested decrypted file to Cockroach Labs.

## Subcommands

While the `cockroach debug` command has a few subcommands, users are expected to use only the [`zip`](/docs/v25.2/cockroach-debug-zip), [`encryption-active-key`](/docs/v25.2/cockroach-debug-encryption-active-key), [`merge-logs`](/docs/v25.2/cockroach-debug-merge-logs), [`list-files`](/docs/v25.2/cockroach-debug-list-files), [`tsdump`](/docs/v25.2/cockroach-debug-tsdump), and [`ballast`](/docs/v25.2/cockroach-debug-ballast) subcommands.

We recommend using the [`encryption-decrypt`](/docs/v25.2/cockroach-debug-encryption-decrypt) and [`job-trace`](/docs/v25.2/cockroach-debug-job-trace) subcommands only when directed by the [Cockroach Labs support team](/docs/v25.2/support-resources).

The other `debug` subcommands are useful only to Cockroach Labs. Output of `debug` commands may contain sensitive or secret information.

## Files

The `cockroach debug encryption-decrypt` command will output the decrypted store file to the specified location. If no location is specified, the decrypted contents are sent to`stdout`.
