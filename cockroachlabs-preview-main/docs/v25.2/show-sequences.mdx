The `SHOW SEQUENCES` [statement](/docs/v25.2/sql-statements) lists all sequences in a database.

## Required privileges

No [privileges](/docs/v25.2/security-reference/authorization#managing-privileges) are required to list the sequences in a database.

## Synopsis

<ShowSequences5 />

## Parameters

<table><thead><tr><th>Parameter</th><th>Description</th></tr></thead><tbody><tr><td><code>name</code></td><td>The name of the database for which to list sequences. When omitted, the sequences in the <a href="/docs/v25.2/sql-name-resolution#current-database">current database</a> are listed.</td></tr></tbody></table>

## Example

```sql
> CREATE SEQUENCE sequence_test;
```

```sql
> SHOW SEQUENCES;
```

```text
  sequence_schema | sequence_name
------------------+----------------
  public          | sequence_test
(1 row)
```
